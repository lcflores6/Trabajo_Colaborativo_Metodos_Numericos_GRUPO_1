# Análisis de Teoría de Errores - Métodos Numéricos

Este repositorio contiene la implementación y el análisis de la **Teoría de Errores** utilizando lenguajes de programación **Python** y **MATLAB**. El proyecto ha sido desarrollado de forma colaborativa para la asignatura de Métodos Numéricos.

## 📝 Descripción
El objetivo de este proyecto es calcular y visualizar los diferentes tipos de errores que ocurren en el cálculo numérico, tales como:
* **Error Absoluto:** La diferencia entre el valor real y el valor aproximado.
* **Error Relativo:** El error absoluto normalizado por el valor real.
* **Propagación del Error:** Cálculo de cómo el error de las variables de entrada afecta al resultado final de una función $f(x, y)$.

## 📂 Estructura del Repositorio
* `/python/`: Scripts de implementación en Python (`.py` o `.ipynb`).
* `/matlab/`: Scripts y funciones de implementación en MATLAB (`.m`).
* `/docs/`: Documentación adicional.
